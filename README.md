# BanderaShop-Front
## WEB
https://banderashop.netlify.app/

## Installation
```git
git clone https://github.com/Glubin-yep/bandera-shop-front
```

## При першому засупуску після клонування
```sh
npm install 
```
- Змніти API_URL в ./http/index.js на http://localhost:4000/api

## Для запуску
```sh
npm run start
```
